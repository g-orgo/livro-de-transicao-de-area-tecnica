# Apêndice D — Glossário

**Ambiente virtual (`.venv`)** — Instalação isolada de pacotes Python para um projeto.

**Anotação de tipo** — Informação como `name: str` ou `-> bool` que documenta e permite análise estática.

**Dataclass** — Classe voltada a dados, com métodos comuns gerados automaticamente.

**Fixture** — Contexto reutilizável fornecido a um teste pelo pytest.

**Lockfile** — Registro das versões resolvidas de dependências; neste livro, `uv.lock`.

**MCP** — Model Context Protocol, padrão para expor e consumir ferramentas, recursos e prompts.

**Pydantic** — Biblioteca de modelos e validação de dados em runtime.

**PyPI** — Índice público de pacotes Python.

**RAG** — Retrieval-Augmented Generation: recuperar contexto antes da geração de uma resposta.

**Ruff** — Ferramenta rápida de lint e formatação Python.

**`pyproject.toml`** — Arquivo padronizado de metadados e configuração do projeto.

**uv** — Ferramenta para versões de Python, ambientes e dependências.
